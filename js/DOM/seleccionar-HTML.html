Seleccionar elementos en el HTML

Frecuentemente, vamos a querer manipular elementos HTML con JavaScript. Para ello deberás, antes que nada, encontrarlos e identificarlos.

Hay varias maneras de hacer esto:

  • Encontrar elementos HTML por id.
  • Encontrar elementos HTML por clase.
  • Encontrar elementos HTML por nombre de etiqueta.
  • Encontrar elementos HTML por selectores de CSS.
