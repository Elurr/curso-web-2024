DOM (Document Object Model)

Con el DOM, JavaScript puede acceder y cambiar todos los elementos de un documento HTML.

Cuando el navegador carga una web, crea un DOM de la página.

El modelo DOM es construído como un árbol jerárquico de objetos:

Con esto, JavaScript obtiene la posibilidad de crear HTML dinámicamente:

  • Puede cambiar todos los elementos de una página.
  • Puede cambiar los atributos.
  • Puede cambiar los estilos CSS.
  • Puede añadir o remover elementos y atributos HTML.
  • Puede reaccionar a todos los eventos HTML de la página.
  • Puede crear nuevos eventos.

¿Qué es el DOM HTML?

Es el modelo de objeto estándar y la interfaz de programación para HTML. Define:

  • Los elementos HTML como objetos.
  • Las propiedades de los elementos.
  • Los métodos para acceder a dichos elementos.
  • Los eventos para todos los elementos (Un ejemplo sería pulsar un botón).

Métodos del DOM

Son acciones que puedes realizar sobre los elementos HTML (como borrar o crear un elemento).

Las propiedades son valores que puedes establecer u obtener de los elementos (como el href de un link).
