El objeto document

Es el dueño o el ancestro de todos los demás objetos de la página web.

Si quieres acceder a cualquier elemento HTML, debes empezar accediendo al objeto document.

Algunos ejemplos para acceder y manipular el HTML:

Encontrar elementos

  • document.getElementById('id'): Encuentra un elemento buscando su id
  • document.getElementsByTagName('etiqueta'): Encuentra elementos buscando su nombre de etiqueta y nos retorna un array.
  • document.getElementsByClassName('clase'): Encuentra los elementos que tengan la clase indicada. Retorna un array (Recordemos que los elementos pueden tener más de una clase).

Manipular elementos

Propiedades

  • elemento.innerHTML = 'nuevo contenido': Reemplaza el contenido de un elemento HTML.
  • elemento.atributo = 'nuevo valor': Cambia el atributo de un elemento (por ejemplo, el atributo type de un input).
  • elemento.style.propiedad = 'nuevo estilo': Cambia el estilo CSS de un elemento.

Método

  • elemento.setAttribute(atributo, valor): Cambia el atributo de un elemento.

Crear y eliminar elementos

Métodos

  • document.createElement('elemento'): Crea un elemento HTML.
  • document.removeChild('elemento'): Elimina un elemento HTML desde el padre.
  • document.appendChild('elemento'): Crea un elemento dentro de otro.
  • document.replaceChild(nuevo, viejo): Sustituye un elemento por otro.
  • document.write(texto): Escribe en el output final de la página (puede sobreescribir toda la página)
