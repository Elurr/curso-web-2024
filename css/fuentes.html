Fuentes

Elegir una fuente apropiada tiene un fuerte impacto en cómo los lectores experimentan un sitio web.

La fuente correcta puede crear una reconocible entidad para tu marca o página web

Usar una fuente fácil de leer es muy importante. Ya que añade valor a tu texto. También es importante elegir un correcto color y tamaño.

Familias de fuente genéricas

En CSS hay 5 familias genéricas:

  1. Fuentes con serif tienen un pequeño trazo en las puntas de cada letra
  2. Fuentes sans-serif. No tiene los trazos de las anteriores y tienen un aspecto minimalista y moderno.
  3. Monospace. Todas letras tienen la misma anchura. Crean un aspecto mecánico. La fuente que vemos en nuestro código es de este tipo.
  4. Cursivas. Imitan la caligafría humana.
  5. Fantasy. Tienen un aspecto decorativo/juguetón.

font-variant

Con esta propiedad podemos especificar que un texto se muestre en "small-caps".

Con esta fuente, todas las minúsculas de un texto se convertirán mayúsculas pero de menor tamaño.

Tamaño de fuente

La propiedad font-size especifica el tamaño de un texto.

Ser capaz de modificar los tamaños de los diferentes textos de una web es muy importante. Sin embargo, no uses párrafos con tamaños mayores para que parezcan títulos y viceversa. Usa las correspondientes etiquetas h1-h6 para títulos o p para párrafos.

Los valores de tamaño pueden ser absolutos o relativos:

Valor absoluto:

  • Establece el texto a un tamaño específico.
  • No permite que el usuario cambie el tamaño del texto desde su navegador (malo por razones de accesibilidad)
  • Un valor absoluto es útil cuando se conoce el tamaño físico del dispositivo con el que se visualizará nuestra página.

Valor relativo:

  • Establece el tamaño relativo a los elementos de alrededor.
  • Permite al usuario cambiar el tamaño del texto en el navegador

Valores em

Para permitir que el usuario pueda redimensionar el texto, los devs o desarrolladores, utilizamos em en lugar de píxeles.

1em equivale al tamaño por defecto del texto que son 16px.

Para calcular la equivalencia de píxeles a em usa esta fórmula: px/16=em

Fuente responsive

Podemos usar la unidad rem para conseguir directamente el efecto de usar la combinación de em y body {font-size: 100%}.

Otra medida responsive es la unidad vw, que significa "viewport width" o ancho de la pantalla.

Texto responsive

Cambia el tamaño de la ventana para ver el efecto

Usa la unidad "vw" para darle tamaño al texto. Por ejemplo, 10vw pondrá el texto al 10% del tamaño del ancho de la pantalla.

Viewport es el tamaño de la ventana del navegador. 1vw equivale al 1% del ancho de la ventana. Si la ventana mide 50cm de ancho, 1vw será 0.5cm de ancho
