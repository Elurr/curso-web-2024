Selectores en CSS

Los selectores se usan para "encontrar" los elementos HTML a los que queremos otorgar de estilo.

Tienen 5 categorías:

  • Selectores simples: nombre, clase, id.
  • Selectores combinadores: Seleccionan elementos basándose en una relación específica entre ellos.
  • Selectores de Pseudo-clase: Seleccionan elementos basándose en un estado específico.
  • Selectores de Pseudo-elementos: Seleccionan y estilizan ciertas partes de un elemento.
  • Selectores de atributo: Seleccionan elementos basándose en un atributo o en el valor de un atributo.

Selector id

Selecciona un único elemento básandose en el atributo id.

El atributo id es único por cada elemento. Si en el documento hay dos elementos con la misma id, ésta se aplicará solamente al último (orden de lectura).

Selector de clase

Selecciona un grupo de elementos basándose en el atributo clase.

Selector universal

Selecciona TODOS los elementos del documento. El selector es un asterisco *

Selectores combinadores

Un combinador es un selector que explica la relación entre dos elementos.

Hay cuatro tipos distintos de combinadores:

  • Selector descendiente (espacio)
  • Selector child o hijo (>)
  • Selector adyacente o hermano (+)
  • Selector de hermano general (~)
