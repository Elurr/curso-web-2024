Colores en CSS

Hay tres maneras de definir colores en CSS: por nombre, RBG y Hexadecimal.

Por nombre:

CSS tiene una lista de colores predefinidos que podemos utilizar simplemente al nombrarlos en el código. Ej: dodgerblue

Por RGB

RGB significa Red, Green, Blue. Cada valor va de 0 a 255 y los colores se consiguen combinando al rojo, verde y azul. Ej: rosa

Hexadecimal

Hexadecimal se representa con una almohadilla # seguida de 6 números (#RRGGBB). Ej: verde
