Introducción a HTML

¿Qué es HTML?

HTML signifca Hyper Text Markup Language. En castellano, lenguaje de marca de hipertexto.

HTML describe la estrutura de una página web. Consiste en una serie de elementos que indican al navegador cómo mostrar la página.

Los elementos marcan los trozos de contenido y lo describen como "esto es un párrafo", "esto es un título", "esto es una imagen/link/botón"...

¿Qué es un elemento?

Estructura de un elemento

Contenido

Un elemento es todo aquello desde la etiqueta de apertura hasta la etiqueta de cierre. Casi todos los elementos están formados por etiqueta de apertura y cierra salvo excepciones.

Ejemplo:
La etiqueta "br" es una etiqueta vacía, es decir, no tiene contenido ni etiqueta de cierre. Sirve para indicar un salto de línea.
