Atributos en elementos de HTML

Todos los elementos pueden contener atributos. Los atributos proveen información adicional sobre los elementos.

Los atributos se especifican en la etiqueta de apertura. Se componen de dos partes, nombre y valor.

name="value"

href

Es el atributo básico de los links (a). Las (a) define un hipervínculo, el atributo href define la URL de la página donde nos va a llevar el hipervínculo.

src

La etiqueta (img) se usa para insertar una imagen en una página HTML. El atributo src indica el camino o dirección de la imagen a mostrar.

src con URL o vínculo absoluto.

src con URL o vínculo relativo.

width y height

Los atributos width, ancho y height, altura, definen el tamaño que queremos otogar a la imagen en nuestra página.

alt

El atributo alt definde el texto alternativo para una imagen. Este texto se mostrará si por alguna razón, la imagen no puede cargarse.

style

Este atributo se usa para añadir estilos visuales a cualquier elemento. Como por ejemplo, color, fuente -font-, tamaño -size-, etc.

Clic para más detalles sobre el style

title

Este atributo se usa para proveer información extra sobre un atributo.

Esta información se muestra en forma de tooltip al dejar el cursor sobre este elemento.
